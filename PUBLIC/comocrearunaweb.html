Bienvenido a Como Crear Una Página Web

En este sitio encontrarás información para crear tu propia página web desde 0.

Visual Studio Code

Para que sea más intuitivo el proceso es recomendable el empleo de Visual Studio Code. Enlace de descarga: Visual Studio Code

Recomendaciones

  • Tener descargado Git
  • Crear un repositorio en GitHub
  • Enlace de descarga: Visual Studio Code

Primeros Pasos

Descubre como empezar.

Creamos un nuevo archivo html

Dentro de Visual Studio Code creamos un archivo html, por ejemplo, lo denominamos "pagina.html".

Estructura de la página

Utilizamos la plantilla básica html, donde tenemos dos apartados. Respecto al primer apartado "head", se trata donde desarrollamos el back-end de la página, es decir, el código que no se muestra. En cuanto al segundo apartado "body", es el front-end de la página, es decir, el código que se muestra al usuario.

Diseño de la página

Para que tenga un estilo la página, utilizamos CSS. El CSS lo podemos incluir de 3 maneras. Mediante un archivo externo, mediante un archivo interno en el head, o mediante en línea en el body.
